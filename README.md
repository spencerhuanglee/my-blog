# Blog

Write Markdown. Push. Live on [spencerhlee.com](https://spencerhlee.com).

## New post

```bash
npm run new -- "My post title"
```

Edit `src/posts/your-slug.md`, then:

```bash
git add .
git commit -m "Add my post"
git push
```

GitHub builds and publishes automatically.

## Preview locally

```bash
npm install
npm start
```

Opens a local preview (usually http://localhost:8080).
